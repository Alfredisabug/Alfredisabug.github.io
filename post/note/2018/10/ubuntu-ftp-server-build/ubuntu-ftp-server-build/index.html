Ubuntu 建立FTP伺服器

Publish Date : 2018-10-09 16:23

之後會有架站的需求
所以開始練習了Linux(之前就有些概念了
我先在虛擬機上灌了Ubuntu
要跟windows本機傳資料
根據之前的經驗
用VM用了共用資料夾之後的設定
Ubuntu都會無法開機(我也不知道為什麼
最後我想到
之前上課有用過FTP
不如試試看吧

我使用的是vsftpd
網路上教學也很多

先下載vsftpd

$sudo apt-get install vsftpd

之後更新 ftp config 檔案

$sudo vi /etc/vsftpd.conf

我只有動到1個設定

write_enable=YES

因為要能夠把windows本機資料傳給Ubuntu
其他的就沒動預設就好

最後啟動

$/etc/init.d/ssh start

之後如果有更動config檔案，要記得重啟

$/etc/init.d/ssh restart

Ubuntu這邊則是需要知道ip位址來給windows連線

要先下載net-tools

$sudo apt-get install net-tools

之後查詢ip address

$ifconfig


知道ip address後
再回到windows本機
我用filezilla連去Ubuntu
下載client版本
安裝好之後，就可以快速連線拉

至於虛擬機的區域網路設定
看網路上大神文章是說
對外跟對內的網卡分開
因此我還有在虛擬機開一張虛擬網卡
並指定address為192.168.197.3
就是如圖上的ip位址

開啟VM


之後新增並設定好

設定好之後，新增介面卡給虛擬機使用

這樣就大功告成了!
