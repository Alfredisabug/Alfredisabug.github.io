UI 免費圖庫

Publish Date : 2018-03-10 13:41

在設計UI時 難免會需要一點靈感
或是需要自己設計icon 圖標 按鈕圖片等
以下推薦幾個網站

設計靈感

Dribbble:
https://dribbble.com/
Behance:
https://www.behance.net/
Pinterest:
https://www.pinterest.com/
其中Dribbble是我比較常看的
裡面包含了很多大師的作品
主要色調部分都附有色碼
可以讓UI工程師在設計面板時做一個參考用

免費圖庫網站:
IconArchive:
http://www.iconarchive.com/
clker:
https://www.clker.com/
FLATICON:
https://www.flaticon.com/

上面是一些沒有版權的圖庫網站
如有錯誤請告知
裡面有各種的圖，可以下載.ico 或是 .png可以選擇
clker也可以自行畫，其他的我比較常下載圖庫
當然如果真的要好看的話
還是推薦使用adobe 的 adobe photoshop

要成為好的UI工程師
就是多看別人的作品
或是看會別人語法就能學習到更多
現在有很多CSS語法或是html語法的網站
像是我用Qt寫UI
就會需要會QSS
QSS又是類似CSS語法
假設在網路上看到某CSS語法
就可以複製看看，就又學到一項

目前也有自己設計的小技巧:
字通常我都會選擇白字(如果背景不是白的話)
空白記得留多，不然會太有壓迫感

色系通常目前是參考別人網站
https://color.adobe.com/zh/create/color-wheel/
這個adobe color網站可以讓我們參考

最後
雖然自己不是學美術出身
或是數媒系等相關科系
但是平常還是要多自己嘗試畫畫看圖片
能讓自己成為更好的UI設計師
